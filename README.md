
 Prerequisites

- [Node.js](https://nodejs.org/) 
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

   Installation
1. Clone the repository:
    git clone <repository-url>
2. Navigate to the project directory:
    cd project-name
3. Install dependencies:
  npm install

  Running the Project
  npm start
